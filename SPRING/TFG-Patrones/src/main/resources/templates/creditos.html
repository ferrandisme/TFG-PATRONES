Informacion


Se plantea diseñar una aplicación que facilite el aprendizaje y la aplicación e implementación de patrones de diseño. Esta aplicación acompañara en las tareas de aprendizaje, eleccion, verificacion, pruebas y validación. Se ha desarrollado utilizando como tecnologias principalmente SPRING, bootstrap y MongoDB. En el apartado metodologico se ha utilizado metodologias agiles apoyadas en la plataforma Worki. El control de versiones se ha llevado a cabo gracias a git y github. Por ultimo, para la gestion de documentos se ha utilizado Microsoft Teams y como IDE IntelliJ y Visual Studio Code.
